# 🚀 Web Development SIG 2026 | PICT CSI Student Branch

This repository contains the code 💻, resources 📚, and reference materials for the Web Development Special Interest Group (SIG) conducted by the PICT CSI Student Branch from March 19 to April 1, 2026. 📅

## 🎯 Overview
The SIG guided 100+ participants 🧑‍💻 through full-stack web development using the MERN stack 🌐. The curriculum progressed from foundational frontend technologies to backend integration and version control ⚙️.

## 🛠️ Technologies Covered
* 🎨 **Frontend Basics:** HTML, CSS, JavaScript, Tailwind CSS
* ⚛️ **Frontend Architecture:** React (Components, Hooks, Routing, Event Handling)
* 🖥️ **Backend & Server:** Node.js, Express.js, RESTful API Design
* 🗄️ **Database:** MongoDB
* 🐙 **Version Control:** Git & GitHub

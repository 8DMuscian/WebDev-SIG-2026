1. Datatypes & Variables

Declares variables of different datatypes (number, string, boolean, object, array)

2. Operators & Conditional Statements

Write a program that:

Takes two numbers as input
Prints:
Sum, Difference, Product
Whether the first number is greater, smaller, or equal to the second

3. Loops

Write a program that:

Prints numbers from 1 to 50
Prints only even numbers using a loop

4. Arrays

Write a program that:

Takes an array of numbers
Finds:
Maximum value
Minimum value
Sum of all elements

5. Functions (Basic)

Write a function:

function isPrime(num) { }

6. Advanced Functions (Arrow + Higher Order)

7. Callback Functions

One that squares the number
One that cubes the number

A button and a paragraph

Change text of paragraph

10. Event Loop + Reflow/Repaint Concept (Practical)

console.log("Start");

setTimeout(() => {
  console.log("Inside setTimeout");
}, 0);

Promise.resolve().then(() => {
  console.log("Inside Promise");
});

console.log("End");

Predict and explain output ofabove code/
Modify DOM inside a loop and observe performance (reflow/repaint)
